# Parcel Locker Mobile App

## Tech stack

- React native + Expo
- Animation: React Native Reanimated
- Icons: Expo Vector Icons
- CSS Framework: NativeWind (TailwindCSS)

## How to run the app

1. Install the dependencies (MUST USE `YARN`):

```bash
yarn
#or
yarn install
```

2. Run the project

```bash
yarn start #general
yarn ios #ios
yarn android #android
```
